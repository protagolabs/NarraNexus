bottom of this file rather than reaching out to individuals directly.

resilient to anyone being away.

## Contact

We deliberately don't publish personal email addresses here.
Use the GitHub features that route to the maintainer team directly:

| What you want to report                              | Where to go                                                                                          |
| ---------------------------------------------------- | ---------------------------------------------------------------------------------------------------- |
| **Security vulnerability** (private)                 | [Open a private advisory](https://github.com/protagolabs/NarraNexus/security/advisories/new)         |
| **Code of Conduct concern** (private)                | DM any maintainer listed above through GitHub, or open a private advisory titled `[CoC]`             |
| **Non-urgent project question / design discussion**  | [GitHub Discussions](https://github.com/protagolabs/NarraNexus/discussions)                          |
| **Bug or feature request**                           | [Open an issue](https://github.com/protagolabs/NarraNexus/issues/new/choose) with the right template |

If you have a strong reason to need a direct email and the channels
above don't fit, leave a brief note in a public issue and a maintainer
will reach out privately.
