# team.integrationLogs

## Description
Gets the integration logs for the current team.

## Required scope
`admin`

## Arguments
| name | type | required | description |
|------|------|----------|-------------|
| `app_id` | string | no | Filter logs to this Slack app. Defaults to all logs. |
| `change_type` | string | no | Filter logs with this change type. Defaults to all logs. |
| `count` | string | no | — |
| `page` | string | no | — |
| `service_id` | string | no | Filter logs to this service. Defaults to all logs. |
| `user` | string | no | Filter logs generated by this user’s actions. Defaults to all logs. |

## Example
```python
slack_cli("team.integrationLogs", {})
```
